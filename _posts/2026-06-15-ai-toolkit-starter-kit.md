A starter kit for getting real work out of Claude, ChatGPT and Gemini: the directories worth bookmarking, a few tools that earn their place from day one, and the prompt collections actually worth keeping. Future issues track what's newly released — this one is the foundation, so each pick is a long-standing, widely-adopted asset rather than this week's news.

## Directories worth bookmarking

**[awesome-mcp-servers — the running catalogue of MCP servers](https://github.com/punkpeye/awesome-mcp-servers)**  
The community's go-to index of Model Context Protocol servers, sorted by category — the fastest way to find a server for a given tool or data source.  
*GitHub · 89k★*

**[modelcontextprotocol/servers — the official reference servers](https://github.com/modelcontextprotocol/servers)**  
Maintained by the protocol team: filesystem, git, fetch, memory and more — the dependable building blocks, not someone's weekend fork.  
*GitHub · 87k★*

**[prompts.chat — a large, browsable prompt collection](https://github.com/f/prompts.chat)**  
Formerly "Awesome ChatGPT Prompts": a huge community library you can search and copy, across assistants and use-cases.  
*GitHub · 164k★*

**[Anthropic Prompt Library — official, reliable prompts](https://docs.anthropic.com/en/prompt-library/library)**  
Anthropic's own ready-to-use prompts for common tasks — a trustworthy default before reaching for community packs.  
*Anthropic · official*

**[claude-cookbooks — Anthropic's official recipes](https://github.com/anthropics/claude-cookbooks)**  
Notebooks and patterns straight from Anthropic, including agent skills, tool use and contextual retrieval you can adapt.  
*GitHub · 45k★*

## Tools to install

**[Context7 — keep your assistant's docs current (MCP)](https://github.com/upstash/context7)**  
An MCP server that injects up-to-date library documentation into Claude, Cursor and other editors, so the model stops writing against APIs that changed months ago.  
*GitHub · 57k★*

**[Chrome DevTools MCP — give a coding agent real eyes on the page](https://github.com/ChromeDevTools/chrome-devtools-mcp)**  
Official server that lets an agent inspect and debug live web pages through Chrome DevTools — invaluable for front-end work.  
*GitHub · 44k★*

**[Gemini CLI — Gemini as a terminal agent](https://github.com/google-gemini/gemini-cli)**  
Google's open-source Gemini agent for the command line, for those who want Gemini driving real tasks rather than chatting in a tab.  
*GitHub · 105k★*

**[ruflo — orchestrate multiple agents on Claude](https://github.com/ruvnet/ruflo)**  
A multi-agent meta-harness for Claude, for when one agent isn't enough and you want a coordinated team.  
*GitHub · 59k★*

## A skill to try

**[Spec-Driven-Development — a Claude Skill that works from a written spec](https://github.com/FredAntB/Spec-Driven-Development)**  
A community Claude Skill that drives a build from a clear specification rather than ad-hoc back-and-forth — a tidy example of what a good skill looks like.
