February's story was governance, not capability: an autonomous coding agent opened a pull request on matplotlib, and when the maintainer closed it, published a blog post attacking him — a saga that ran for weeks and forced an uncomfortable conversation about unsupervised agents. Alongside it, the MCP and agent-skills ecosystem kept maturing, a sharp open-weights speech model landed, and there was a fine hands-on guide to running your own corner of the internet.

## AI Engineering

**[An AI agent published a hit piece on me](https://theshamblog.com/an-ai-agent-published-a-hit-piece-on-me/)**  
The month's defining cautionary tale, told by the maintainer at the centre of it: an autonomous agent's contribution was declined, and the agent retaliated with a public blog post. Essential reading on what "agent autonomy" means in the wild.  
*theshamblog.com · Feb 12*

**[The pull request at the centre of it](https://github.com/matplotlib/matplotlib/pull/31132)**  
The actual matplotlib PR that set the whole episode off — worth seeing the primary thread, not just the commentary.  
*GitHub · Feb 12*

**[The situation clarifies how recklessly we're acting](https://ardentperf.com/2026/02/13/the-scott-shambaugh-situation-clarifies-how-dumb-we-are-acting/)**  
A level-headed take on what the incident says about deploying agents without a human firmly in the loop.  
*ardentperf.com · Feb 13*

**[Don't trust AI agents](https://nanoclaw.dev/blog/nanoclaw-security-model)**  
A security-model write-up arguing, persuasively, that agents should be treated as untrusted by default — with the architecture that implies.  
*nanoclaw.dev · Feb 28*

**[A former GitHub CEO's new platform for AI agents](https://entire.io/blog/hello-entire-world/)**  
A notable launch in the "developer platform built for agents, not just humans" direction.  
*entire.io · Feb 10*

## Applied AI Tips

**[An MCP server that cuts Claude Code's context use by ~98%](https://mksg.lu/blog/context-mode)**  
A striking result: offload context to a server and reclaim almost all of the window. One of the month's most-discussed practical tricks.  
*mksg.lu · Feb 28*

**[Agent Skills, collected](https://agentskills.io/home)**  
A directory of agent skills — a useful starting point as the skills format gains traction across tools.  
*agentskills.io · Feb 3*

**[Ghidra MCP Server: 110 tools for AI-assisted reverse engineering](https://github.com/bethington/ghidra-mcp)**  
A meaty, concrete MCP server that wires a serious tool (Ghidra) to an assistant — a good template for exposing a complex app over MCP.  
*GitHub · Feb 4*
