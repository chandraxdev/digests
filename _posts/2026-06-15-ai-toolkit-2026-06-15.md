Weighted where you want it: reusable prompts and skills. The skills below genuinely shipped this fortnight (date-tracked); the prompt collections are evergreen staples worth keeping, marked as such rather than dressed up as new.

## Skills shipped this fortnight

**[prompt-refine-skill — a skill that rewrites your prompt for you](https://github.com/Li-Bailiang/prompt-refine-skill)**  
An Agent Skill that quietly refines whatever you type for the model currently running — a rare skill that is itself *about* better prompting.  
*GitHub · created Jun 5*

**[loop-engineering — patterns and starters for agent loops](https://github.com/cobusgreyling/loop-engineering)**  
Practical patterns, starter templates and CLI tools for "loop engineering" an agent, from a prolific LLM practitioner (Cobus Greyling).  
*GitHub · created Jun 9*

**[ponytail — make your agent think like a blunt senior dev](https://github.com/DietrichGebert/ponytail)**  
A skill that gives an agent the instincts of a no-nonsense senior engineer — it went viral this week (thousands of stars in days), which says something about the appetite for opinionated defaults.  
*GitHub · created Jun 12*

**[vibe-check — pressure-test an idea before you build it](https://github.com/TexasBedouin/vibe-check)**  
A skill from a long-time product manager that interrogates a beginner's product idea and surfaces the holes early.  
*GitHub · created Jun 5*

## Prompts worth keeping (evergreen)

**[Fabric — a library of ready-to-run prompt "patterns"](https://github.com/danielmiessler/Fabric)**  
An open-source collection of curated prompts for everyday tasks — summarise, extract, analyse, write — that you can run from the command line or lift into any assistant. The single best starting point for reusable prompts.  
*GitHub · 42k★ · evergreen*

**[The system prompts behind the major AI tools](https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools)**  
The actual system prompts of Claude Code, Cursor and the rest, collected in one place — less a prompt to copy than a masterclass in how the best tools instruct a model, well worth borrowing from.  
*GitHub · 140k★ · evergreen*
